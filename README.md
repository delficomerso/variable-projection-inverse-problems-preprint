# Variable Projection Methods for Solving Regularized Separable Inverse Problems

## Preprint

This repository corresponds to a preprint based on the work developed for my
undergraduate thesis (Licenciatura en Ciencias de Datos):

- **D. B. Comerso Salzer**,  
  *Variable Projection Methods for Solving Regularized Separable Inverse Problems with Applications to Semi-Blind Image Deblurring*,  
  arXiv:2601.05224  
  https://arxiv.org/abs/2601.05224

## Code availability

The code used in this work is available in the repository developed and maintained by my
thesis advisor:

- **RGenVarPro**  
  https://github.com/malenaespanol/RGenVarPro

This repository does not duplicate code and is intended solely as a reference entry point
for the associated preprint.
